How it started

It was one summer day in my hometown when a friend suggested that it would be cool if we could make our own game and just do whatever we wanted in it. Many plans were made, stories were built, destroyed and built again and while game development never properly started i decided that game development is something i want to be a part of. And so adventures in Unity game engine began, many project scrapped, scripts coded and eventually deleted, but despite all this not going anywhere, experience was built.
